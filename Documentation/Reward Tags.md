# Reward Tags Documentation

The **Reward Tags** system is a universal gamification engine designed to recognize and celebrate user progress across various features of the app (Daily Tasks, Weekly Tasks, Long-term Goals, etc.). It uses a tiered system based on performance metrics like consistency, quality (rating), and time commitment.

## ⚙️ How it Works: Calculation Logic

The system uses a unified calculation engine in `RewardManager`. While each feature (Source) generates its own raw metrics, they are all mapped to a set of core parameters used to determine the earned tier.

### Core Calculation Parameters
- **Progress (%)**: How much of the planned work was completed.
- **Rating (1-5★)**: The qualitative assessment of the work.
- **Points Earned**: Raw points generated by the activity.
- **Completed Days**: Count of successful days in a streak or duration.
- **Consistency (%)**: Ratio of completed days to total scheduled days.
- **Hours/Day**: Average time commitment.
- **Task Stack**: Multiplier for consecutive weekly completions.
- **On-Time Completion**: Reward penalty or disqualification if overdue.

## 🧊 Source-Specific Logic

Rewards are calculated differently depending on where they come from (`RewardSource`).

### 📅 Daily Tasks
- **Bonuses**: Feedback Base, Media, Completion +10, Duration, Priority, On-Time +20.
- **Progress**: `clamp(pointsEarned - totalPenalties, 0, 100)`.
- **Rating**: `1.0 + 4.0 * (progress / 100)`.

### 🗓️ Weekly Tasks
- **Daily Progress**: Rewards calculated for each individual day.
- **Weekly Summary**: Averages metrics across all *active* days. Future empty days do not penalize the current average.

### 🎯 Long-term Goals
- Rewards based on completed milestones and total points.
- Points awarded for milestone bonuses (+20 per 25%) and completion (+40).
- Penalty for inactivity (-10 per 3 days).

### 📝 Bucket List
- Rewards based on checklist completion and streaks.
- Streak bonus (+10 per 3 days) and completion bonus (+20).
- Overdue penalty (-20) and no-media penalty (-10).

## 🎨 User Experience: Reveal & Display

### 🎁 Reward Box & Scratch Card
When a reward is earned, it appears as a glowing **PremiumRewardBox**. 
- Tapping it opens the **PremiumScratchCardPopup**.
- Users must scratch (drag) away the surface to reveal the reward.
- Reveal triggers at **32% coverage** for a satisfying feel.
- High-tier rewards (Crystal+) trigger intense confetti and particle effects corresponding to the tier's color.

### 🤖 Dynamic Feedback
The `RewardTextEngine` generates personalized "Tag Reasons" and "Suggestions".
- **Tag Reason**: Explains exactly why the tier was reached (e.g., "Masterful consistency! 30 days, 3 h/day...").
- **Suggestion**: Encourages the user to reach the next tier by showing the missing requirements (e.g., "Aim for 97%+, 4.7★... for Radiant Force!").

### ✨ Animated Emojis
All rewards are displayed using the `AnimatedRewardEmoji` widget, which adds subtle breath and pulse animations to the emojis, making them feel "alive" on the dashboard.
